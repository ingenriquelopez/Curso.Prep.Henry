-Variables: Las variables son espacios que les asignamos un nombre donde podemos guadar informacion, es como un cajon que puede guardar ropa y le ponemos por fuera "CajonDeRopa" o un cajon donde guadamos juguetes y le ponemos por fuera "CajonDeJuguetes"

-Strings: Son un tipo de dato que puede almacenar una variable, las variables de tipo String pueden almacenar texto,letras o numeros, por ejemplo:

-Funciones(argumentos,return)
Las funciones son pedazos de codigo, que estan agrupados y a este grupo le podemos llamar de alguna manera. Es una lista de sentencias que estan entre llaves y a todo lo que esta entre llaves le asignamos un nombrel, por ejemplo:

function nombrecompleto() {
	var nombre   ="Enrique"
	var apellido = "Lopez"
	console.log(nombre+ apellido)
}
Las funciones despues de hacer algo con lo que mtienen adentro, pueden devolver un valor con la palabra return.

function nombrecompleto() {
	var nombre   ="Enrique"
	var apellido = "Lopez"
	var ncompleto = nombre + apellido

	return ncompleto
}
Tambien existe algo llamado argumentos, los argumentos son valores que le enviamos a la funcion para que haga algo con ellos, van despues del nombre de la funcion y encerrados entre parentesis.

function nombrecompleto(nombre,apellido) {
	var ncompleto = nombre + apellido

	return ncompleto
}
para llamar a una funcion como la anterior y enviarle valores de argumentos, seria asi:
var persona = nombrecompleto('Enrique','Lopez')

	console.log('Es mayor de edad')
